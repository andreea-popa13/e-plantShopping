# e-plantShopping
## Paradise Nursery Shopping Application

This project, **e-plantShopping**, is a React-based web application developed for the IBM Front-End Developer Professional Certificate. It simulates an online nursery where users can browse a variety of houseplants and manage a shopping cart.

### Key Features:
* **Product Listing:** A collection of unique houseplants categorized by type.
* **Shopping Cart:** Add, remove, and update quantities of plants using Redux.
* **Interactive UI:** A landing page with a "Get Started" button and a responsive navbar.

### Technology Stack:
* React (Vite)
* Redux Toolkit
* CSS3
